安装 deb 类型软件 
【安装】deb 类型文件
      sudo dpkg -i file.deb（如果是red hat linux的rpm格式，用rpm -ivh file.rpm）
      dpkg -i  安装一个 Debian 软件包。
      dpkg -r  移除一个已安装的包裹。
      dpkg -I  从中提取包裹信息。
      dpkg -c  列出的内容。
      dpkg -r  移除一个已安装的包裹。
      dpkg -P  完全清除一个已安装的包裹。和 remove 不同的是，remove 只是删掉数据和可执行文件，purge 另外还删除所有的配制文件。
      dpkg -s  显示已安装包裹的信息。同时请看 apt-cache 显示 Debian 存档中的包裹信息，以及 dpkg -I 来显示从一个 .deb 文件中提取的包裹信息。
